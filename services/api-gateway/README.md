# API Gateway

Reverse proxy for auth, content, payments, and analytics services.

## Run

```bash
npm install
npm run dev
```

## Environment

- PORT
- CORS_ORIGIN
- AUTH_SERVICE_URL
- CONTENT_SERVICE_URL
- PAYMENTS_SERVICE_URL
- ANALYTICS_SERVICE_URL

## Exposed Paths

- /api/auth
- /api/users
- /api/subscriptions
- /api/rentals
- /api/partners
- /api/content
- /api/live
- /api/ads
- /api/downloads
- /api/campaigns
- /api/analytics
