# Frontend

Next.js app for browsing, playback, admin tools, and social discovery.

## Run

```bash
npm install
npm run dev
```

## Environment

- NEXT_PUBLIC_API_BASE_URL
- NEXT_PUBLIC_GA4_ID
- NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY
- NEXT_PUBLIC_RAZORPAY_KEY_ID
